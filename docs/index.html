Discord Leveling System

Commands

ping Starts an ultra rapid fire table tennis match.
help Shows a list of commands.
profile Displays your profile.
toplist Displays the servers player ranking.

Experience

Experience is rewarded to users contributing in voice channels of a server. For each minute a user spends inside a voice channel, one experience is rewarded to this user. The accumulated experience value of a user determines his level. Therefore, users with a higher level spend more time participating in active voice communication on discord servers.
